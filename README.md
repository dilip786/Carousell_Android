# Carousell_Android

 
### Libraries Used

* Kotlin extensions
* Retrofit for network calls.
* Rxjava2 for asynchronous calls.
* Junit/Expresso/Mockito for Unit/UI tests
* Glide for image loading.

### Design Patterns

* MVVM + Clean architecture.
